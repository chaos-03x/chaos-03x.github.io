Github Action

这是一个自动化工作流工具，用于实现 CI/CD (持续集成/持续交付或部署)。它通过事件触发（如push或pull request）自动化CI（continuous integration）和CD（continuous deployment），用于在代码仓库中构建、测试、部署和监控软件项目。
